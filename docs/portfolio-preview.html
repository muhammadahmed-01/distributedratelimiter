Production lessons

  • Redis down: fail-closed (503) by default; fail-open is explicit opt-in
  • Clock skew: SWL uses wall-clock timestamps; NTP on app and Redis hosts
  • Hot keys: shared NAT or viral accounts concentrate on one Redis key
  • Filter order: IP exhaustion blocks JWT traffic (proven in k6 combo test)
